## Core Technology: AI Skin Analysis Tool

### Key Capabilities
The AI Skin Analysis Tool uses **180° face mapping** to scan and measure skin types and conditions. It provides detailed examination on skin age and diagnoses skin concerns to enable customized skincare routines.

### Analysis Modes

| Mode | Description |
|------|-------------|
| **HD Skin Analysis** | Ultra-Precise Skin Analysis with Detailed Results by Each Area. Algorithm trained on photos with twice the resolution of standard definition. |
| **SD Skin Analysis** | Overall Skin Examination. Basic and reliable evaluation of skin's overall condition. Offers both photo and live camera modes. |
| **Photo Diagnostic** | Analyze skin from uploaded photos |
| **Live Diagnostic** | Real-time skin analysis via camera |

## Key Features

### 180° Full Face Mapping
Analyzes three facial images (front, left, right profiles) for comprehensive skin analysis, detecting concerns in difficult-to-map areas like chin and cheeks.

### AI Skincare Recommender
Combines skin analysis with product recommendation. Scans and analyzes product data (names, descriptions, images, categories) to generate precise suggestions aligned with customer's unique skincare needs.

### AI Skin Simulation
Creates before/after skin simulations on user's face to visualize and track progress of skin issue treatments. Can generate results for up to 7 skin issues.

### Real-Time AR Overlay
Generates real-time AR overlay to visualize findings, providing interactive skincare shopping experience.

### Skin Age Estimation
Provides detailed examination on skin age with 15-degree wider angles.
